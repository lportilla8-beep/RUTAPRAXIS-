# RutaPraxis

Aplicación para la gestión y seguimiento de prácticas pedagógicas mediante el registro de bitácoras, evidencias y evaluaciones.

## Tecnologías utilizadas

- Java
- Oracle Database
- JDBC
- GitHub

## Funcionalidades

- Inicio de sesión
- Registro de actividades
- Gestión de evidencias
- Seguimiento de prácticas
- Evaluación de estudiantes
